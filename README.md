Cyber-security-awareness-notes
A mini project which encourages and promote  cyber security awareness among secondary school students.Created for C5- ISC -23: Introduction to computer cyber security.It focuses on three key areas of cybersecurity awareness:
1. Email Safety
2. Password Hygiene
3. Safe Browsing

It contains:
- Cyber-security-Awareness-Notes.docx – A summary document highlighting best practices.
- SafeBrowsingTips.txt – Simple guidelines on safe internet use.
- README.md – Project overview and author details.
